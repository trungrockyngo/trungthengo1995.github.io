Camera Canvas's repo

Camera Canvas: Desktop Application of Photo Tailoring for People with Disablities

Description: A desktop application written in C# with , Camera Canvas helps the people with motor impairment to have more accessible and enjoyable experiences of editing uploaded photo by different features of image filtering and drawing simplisitic photos as well as sharing with other users. People with Disablities uses Camera Canvas the best way with the integration of Camera Mouse, the video-input interface using facial features tracking, so they can interact with Camera Canvas by their eyes, nose, etc.
Technologies: C# with the approach of Object Oriented Design, .NET and EmGU
Comments: I received the LEEP Project scholar funding to work as the research assistant in Department of Computer Science in Boston University and developed new features to the existing desktop application, Camera Canvas while conducting some research methods to doing the testing for the application.

Related repo

Simple Speech Recognizer with Gausssian Mixture Model

Description: A command-line program written in Python - NumPy, the program encompasses the implementation of Gaussian Mixture Model to the general version and apply for some audio recognition testing of single phonemes, whose tests comprise of a set of .wav files to be parsed into feature files (.feat) as testing input
Technologies: Python, NumPy, PyLab
Comments: This is the two-week final project during my Machine Learning class at Clark University, where I sought for some speech recognition idea with Professor Basye and was able to settle down for the probalistic learning model of Gaussian Mixture Model. The implementation used similar implementation from the tool, CMUSphinx, in order to get the scripting of audio parsing done.
